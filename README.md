# Jeet Portfolio (Next.js + Tailwind)

Dark-theme portfolio built from your resume.

## Local run

```bash
npm install
npm run dev
```

## Deploy to Vercel

### Option A — Vercel CLI (fastest)

```bash
npm i -g vercel
vercel login
vercel deploy
```

### Option B — GitHub integration

1. Push this repo to GitHub
2. In Vercel: **Add New → Project → Import Git Repository**
3. Framework preset: **Next.js** (auto-detected)
4. Click **Deploy**

## Customize

Edit `app/page.js`:
- profile (name, phone, email, links)
- projects list (add real links/screenshots)
- skills/certifications
